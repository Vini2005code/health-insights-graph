Sobre o projeto

Este projeto foi desenvolvido como um laboratório prático para explorar como dados podem ser estruturados e transformados em soluções funcionais usando ferramentas low-code.
O foco não está no domínio médico, mas sim na construção de lógica, organização de dados e criação de base para aplicações rápidas.

 Motivação

O projeto surgiu da minha necessidade de evoluir em low-code development e construção de produtos com IA.
Quis entender, na prática, como dados podem ser organizados e utilizados dentro de fluxos modernos de desenvolvimento, especialmente em ferramentas como Lovable, voltadas para criação rápida de aplicações.
Esse projeto representa um passo direto na minha transição para trabalhar com construção de produtos digitais usando IA e automação.

  O que eu pratiquei
- Estruturação de dados para aplicações reais
- Lógica de produto aplicada a datasets simples
- Construção de fluxos low-code com foco em velocidade
- Uso de ferramentas de IA para prototipação e desenvolvimento
 Para recrutadores (Lovable / AI tools / low-code)
- Este projeto demonstra minha capacidade de aprender rápido, estruturar dados e construir soluções funcionais com ferramentas low-code.
- Meu foco é atuar em ambientes onde IA, automação e construção de produtos rápidos são parte do dia a dia, como o ecossistema da Lovable.

Observação

Todos os dados utilizados são fictícios e servem apenas para experimentação e aprendizado.
